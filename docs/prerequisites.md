# Prerequisites

Before implementing this project ensure the following components are available.

## AWS

- AWS Account
- IAM permissions
- VPC
- Amazon EKS Cluster
- Amazon ECR

## Local Machine

- Git
- Docker
- kubectl
- AWS CLI
- Terraform
- Java
- Maven

## CI/CD

- Jenkins
- SonarQube
- Trivy
- ArgoCD

## Kubernetes

- kubectl configured
- Ingress Controller
- Metrics Server
